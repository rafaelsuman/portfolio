The Solution

Built a complete end-to-end BI pipeline that automatically extracts data from Trinks API, transforms it through N8N workflows, stores it in a MySQL data warehouse, and delivers insights via Power BI dashboards and WhatsApp reports.

Data Warehouse Architecture

Constellation Schema Design

  • DIM_CLIENTES: Customer dimension with retention status tracking
  • DIM_PROFISSIONAIS: Professional dimension with performance metrics
  • DIM_SERVICOS: Service catalog with pricing and categories
  • FATO_TRANSACOES: Transaction fact table (header level)
  • FATO_TRANSACOES_SERVICOS: Service line items with proportional revenue attribution
  • FATO_TRANSACOES_PRODUTOS: Product line items
